# Product Scanner Server

## Create virtual environment & dependencies

```bash
sudo apt install libsqlite3-dev
python3.10 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
deactivate
```

## Run

```bash
source .venv/bin/activate
python main.py
deactivate
```
